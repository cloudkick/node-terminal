# node-terminal

A collection of utility functions which come handy when writing terminal /
console applications.

## Features

* Spinners
* Tables
* Question prompts
* Stylized output (colors & styles)

## Compatibility

* Nodejs 0.4
* Nodejs 0.6
* Nodejs 0.8
* Nodejs 0.10
* Nodejs 0.11
